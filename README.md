# VsOS

VsOS is a very early 32-bit x86 operating system built for a retro desktop experience inspired by Windows 2000 and 90s-era PCs.

It is still very raw and incomplete.

## Requirements

| Parameter | Value |
| :-- | :-- |
| CPU | 1 core, x86, 32-bit |
| RAM | 16 MB minimum, 32-64 MB recommended |
| GPU | VESA-compatible (VBE) |
| Storage | Not required, runs from RAM |
| Boot Media | ISO image (about 9 MB) |

## Features

- Retro desktop interface in a Windows 2000-style layout
- Basic window management
- Start menu and taskbar
- `Win + R` hotkey
- Single Run command: `cmd`
- Mouse and keyboard input
- Basic sound support
- Built-in support for:
  - `.txt`
  - `.ini`
  - `.wav`
  - `.bmp`
  - PC speaker

## Notes

- VsOS is still unstable and unfinished.
- Many parts are experimental.
- Some features are minimal or missing.
- It is a hobby OS, not a production system.

## Goal

The goal of VsOS is to create a small, retro-looking operating system with a simple desktop environment and basic multimedia support.
